HTML学习笔记day1

发表于2024-09-15|更新于2024-12-08|前端学习笔记

网页的相关概念

    1. 网页：由HTML、CSS、JavaScript、图片等组成的静态文档，通过浏览器访问。
    1. 网页的组成：HTML：结构、内容、标记语言；CSS：样式表语言；JavaScript：脚本语言；图片：位图、矢量图、动画图。
    1. 什么是HTML？HTML（HyperText Markup Language，超文本标记语言）是一种用于创建网页的标记语言，它是一种基于XML的标准通用标记语言。[是一种标记语言，不是编程语言。]

常用的浏览器

  • Chrome：Google开发的开源浏览器，功能强大，速度快，安全性高。 Firefox：Mozilla开发的开源浏览器，功能强大，速度快，安全性高。Internet Explorer：微软开发的浏览器，功能强大，速度慢，安全性低。Safari：Apple开发的浏览器，功能一般，速度快，安全性高。
  • 浏览器内核:
    • 负责读取网页内容，整理讯息，计算网页的显示方式并显示页面。
    • 浏览器内核的类型：
      • Trident：IE内核，由微软开发，曾经是IE浏览器的主流内核，现已被Edge取代。
      • Gecko：Mozilla基金会开发的开源内核，火狐浏览器、SeaMonkey等使用该内核。
      • Webkit：Apple公司开发的开源内核，Safari浏览器、Chrome浏览器使用该内核。
      • Blink：Google公司开发的开源内核，Chrome、Opera等使用该内核。

Web标准

  • Web标准是一系列的规则、标准和技术，旨在确保互联网上信息的可访问性、可用性、安全性和效率。
  • 目前，W3C组织制定了Web标准，包括HTML、CSS、JavaScript、SVG、MathML、WebGL、WebRTC、XML、XSLT、XForms、ACID、ARIA、Web Accessibility等。

HTML标签

  • HTML标签是用来定义网页内容的，HTML标签分为：

    • 块级标签：div、p、h1~h6、ul、ol、li、table、form、header、footer、section、nav、article、aside、details、dialog、figure、figcaption、video、audio、canvas等。
    • 行内标签：a、span、img、input、select、button、label、textarea、strong、em、i、b、u、s、sub、sup、iframe、embed、object、param、map、area、nav、header、footer、section、article、aside、details、dialog、figure、figcaption、video、audio、canvas等。

  • HTML标签的属性：

    • 全局属性：class、id、style、title、lang、dir等。
    • 事件属性：onclick、ondblclick、onmousedown、onmouseup、onmouseover、onmouseout、onmousemove、onkeydown、onkeypress、onkeyup、onload、onunload、onfocus、onblur、onsubmit、onreset、onselect、onchange等。
    • 表单属性：name、type、value、maxlength、size、checked、disabled、readonlyp等。

  • HTML基本结构标签:

    • 文档类型声明：<!DOCTYPE html>
    • 网页头部：包括<head>标签和<title>标签。
    • 网页主体：包括<body>标签。
    • 注释：

  • 解释一些标签

    • <!DOCTYPE html>：作用是定义文档类型，告诉浏览器使用哪种规范来解析网页。[使整个文档的标签，必须写在第一行，且必须写上。]
    • <html lang="en">：lang属性定义网页的语言.[en表示英文，zh-CN表示中文简体。]
    • <meta charset="UTF-8">：charset属性定义网页的字符集。

  • 常用的标签(根据标签的语义，在合适的地方给一个最为合理的标签，可以使页面更加清晰)

    • <h1>~<h6>：标题标签，用来定义标题，h1表示最重要的标题，h6表示最低的标题。作为标题使用，并且依据重要性递减。加了标题标签的文字，会变粗体。一个标题独占一行。
    • <p>：段落标签，用来定义一个段落。段落标签可以包含多个
      标签，用来换行。
    • <a href="网址">：超链接标签，用来定义超链接，href属性定义链接的地址。
    • <img src="图片地址" alt="图片描述">：图片标签，用来定义图片，src属性定义图片的地址，alt属性定义图片的描述。
    • <ul>、<ol>、<li>：列表标签，用来定义列表。ul表示无序列表，ol表示有序列表，li表示列表项。
    • <table>、<tr>、<td>、<th>：表格标签，用来定义表格。table表示表格，tr表示表格行，td表示表格单元格，th表示表头单元格。
    • <form>、<input>、<select>、<button>：表单标签，用来定义表单。form表示表单，input表示输入框，select表示下拉框，button表示按钮。
    • <div>：块级容器标签，用来定义一个块级容器。div可以用来对网页内容进行分组。独占一行
    • <span>：行内容器标签，用来定义一个行内容器。span可以用来对文字进行分组。[盒子用来装内容]
    • <header>、<nav>、<section>、<article>、<aside>、<footer>：语义化标签，用来定义网页的不同部分。header表示网页的头部，nav表示导航栏，section表示网页的章节，article表示文章，aside表示侧边栏，footer表示网页的底部。
    • <audio>、<video>：音频、视频标签，用来定义音频、视频。audio表示音频，video表示视频。
    • <canvas>：画布标签，用来定义图形。canvas表示一个画布，可以用来绘制图形。
    • <style>：样式标签，用来定义网页的样式。style表示样式表，可以用来设置网页的字体、颜色、背景等。
    • <script>：脚本标签，用来定义网页的脚本。script表示脚本，可以用来实现网页的动态效果。
    • 文本格式化标签：
      • <b>、<i>、<u>、<s>：加粗、斜体、下划线、删除线。
      • <sub>、<sup>：下标、上标。
      • <small>、<big>：小号字体、大号字体。
      • <center>：居中对齐。
      • <font>：字体标签，用来定义字体。
      • <pre>：预格式化标签，用来定义预格式化文本。
    • 锚点链接标签：
      • <a name="锚点名称">：定义锚点。
      • <a href="#锚点名称">：链接到锚点。
