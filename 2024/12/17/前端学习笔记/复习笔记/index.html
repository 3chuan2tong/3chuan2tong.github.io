HTML复习

  1. HTML是标记语言，CSS是样式语言，JavaScript是脚本语言。
  2. 单标签元素：img、br、input、hr、meta、link
    表单、列表、表格元素：form、ul、ol、li、table、tr、td、th、thead、tbody、tfoot、colgroup、col、select、option、textarea、button、label、input。

CSS复习

  1. CSS选择器：标签选择器、类选择器、ID选择器、属性选择器、后代选择器、子选择器、相邻兄弟选择器、伪类选择器、伪元素选择器。
    权重与优先级：内联样式 > 内部样式表 > 外部样式表 > 继承样式 > 浏览器默认样式。

  2. 元素显示模式:

    • block: 块级元素，独占一行，宽度默认是父元素宽度的100%，高度默认是内容的高度,是一个容器及盒子，里面可以放行内或者块级元素。高度，宽度、外边距以及内边距都可以控制.

      • 文字类的元素内不能使用块级元素
      • <p> 标签主要用于存放文字，因此 <<p> 里面不能放块级元素，特别是不能放<div>
      • 同理， <h1>~<h6>等都是文字类块级标签，里面也不能放其他块级元素

    • inline: 相邻行内元素在一行上，一行可以显示多个，高、宽直接设置是无效的,默认宽度就是它本身内容的宽度。,行内元素只能容纳文本或其他行内元素

    • inline-block: 行内块元素，既可以设置宽度和高度，又可以设置margin和padding。默认宽度是内容的宽度。

    • none: 隐藏元素，不占位置。

    • 转换为块元素：display:block;转换为行内元素：display:inline;转换为行内块：display: inline-block;

  4. CSS三大特性：

    • 继承：子元素继承父元素的样式。
    • 层叠：相同选择器给设置相同的样式，此时一个样式就会覆盖（层叠）另一个冲突的样式。层叠性主要解决样式冲突的问题。样式冲突，遵循的原则是就近原则，哪个样式离结构近，就执行哪个样式
    • 优先级：!important > 内联样式 > 内部样式表 > 外部样式表 > 继承样式 > 浏览器默认样式。:

  5. CSS盒模型：

    • CPBM: content、padding、border、margin
    • 外边距合并:主要有两种情况:
      1. 相邻块元素垂直外边距的合并:当上下相邻的两个块元素（兄弟关系）相遇时，如果上面的元素有下外边距 margin-bottom，下面的元素有上外边距 margin-top ，则他们之间的垂直间距不是两者之和。取两个值中的较大者
      2. 嵌套块元素垂直外边距的塌陷:对于两个嵌套关系（父子关系）的块元素，父元素有上外边距同时子元素也有上外边距，此时父元素会塌陷较大的外边距值。可以为父元素定义上边框,或者为父元素定义上内边距,或为父元素添加 overflow:hidden。

  6. 浮动与清除浮动：

    • 浮动元素：float属性设置为left或right，元素会脱离文档流，并向左或向右移动，直到遇到另一个浮动元素或碰到父元素的边框。浮动的盒子不再保留原先的位置，没有缝隙。
    • 清除浮动：父级没高度、盒子浮动了、影响下面布局了，我们就应该清除浮动了。
      1. 额外标签法也称为隔墙法:额外标签法会在浮动元素末尾添加一个空的标签。例如 <div style=”clear:both”></div>，或者其他标签如<br />等）。
      2. 父级添加overflow属性:给父级添加 overflow 属性，将其属性值设置为 hidden、 auto 或 scroll 。

      4. 父级添加双伪元素:

  7. 定位:将盒子定在某一个位置，所以定位也是在摆放盒子， 按照定位的方式移动盒子。定位 = 定位模式 + 边偏移 。

    • 定位模式用于指定一个元素在文档中的定位方式。边偏移则决定了该元素的最终位置。
边偏移属性 示例 描述
top top: 80px 顶端偏移量，定义元素相对于其父元素上边线的距离。
bottom bottom: 80px 底部偏移量，定义元素相对于其父元素下边线的距离。
left left: 80px 左侧偏移量，定义元素相对于其父元素左边线的距离。
right right: 80px 右侧偏移量，定义元素相对于其父元素右边线的距离
  • static(静态定位):默认定位方式，无定位的意思。没有边偏移
  • relative(相对定位):相对定位是元素在移动位置的时候，是相对于它原来的位置来说.原来在标准流的位置继续占有，后面的盒子仍然以标准流的方式对待它。(相对定位并没有脱标)
  • absolute(绝对定位):绝对定位是元素在移动位置的时候，是相对于它祖先元素来说的。如果祖先元素没有定位，则以浏览器为准定位（Document 文档）。如果祖先元素有定位，那么它就是相对于祖先元素来定位的。(绝对定位会脱标)
  • fixed(固定定位):元素固定于浏览器可视区的位置。不在占有原先的位置(特殊的绝对定位)
  • 定位叠放次序 z-index:选择器 { z-index: 1; }数值可以是正整数、负整数或 0, 默认是 auto，数值越大，盒子越靠上.
  1. 元素的显示与隐藏:
    • display:none; 隐藏元素，不占位置。
    • visibility:hidden; 隐藏元素，占位置，但内容不可见。
    • opacity:0; 透明度，0为完全透明，1为完全不透明。
    • overflow:hidden; 内容溢出隐藏，防止内容溢出盒子。

HTML5和CSS3

  1. HTML5新增的语义化标签:

    <header>：头部标签
    <nav>：导航标签
    <article>：内容标签
    <section>：区域标签,定义文档某个区域
    <aside>：侧边栏标签
    <footer>：尾部标签

  2. HTML5新增多媒体标签:

    <audio>：音频标签
    <video>：视频标签

  3. CSS3新特性
    CSS3 给我们新增了选择器，可以更加便捷，更加自由的选择目标元素。
    • 属性选择器
    • 结构伪类选择器
    • 伪元素选择器

JavaScript复习

  1. 声明变量优先使用const，如果发现它后面是要被修改的，再改为let

对象、数组是引用类型，里面存储的是地址，只要地址不变，就不会报错，建议数组和对象使用 const 来声明
2.

  • DOM: Document Object Model，文档对象模型，是W3C组织推荐的处理网页内容的标准编程接口。操作网页内容可以开发网页内容特效和实现用户交互。

  1. DOM树:将 HTML 文档以树状结构直观的表现出来，我们称之为文档树或 DOM 树可以直观的体现标签与标签之间的关系。

  2. DOM操作:

    • 获取元素:document.getElementById()、document.getElementsByTagName()、document.querySelector()、document.querySelectorAll()
    • 创建元素:document.createElement()
    • 增加元素:appendChild()、insertBefore()
    • 删除元素:removeChild()
    • 修改元素:innerHTML、innerText、setAttribute()
    • 事件监听:addEventListener()、removeEventListener()
    • 操作元素样式属性:元素.className = '类名', 元素.classList.add('类名')、classList.remove('类名')、classList.toggle('类名')
    • 定时器:setTimeout()、setInterval()

  3. 事件类型:

    • 鼠标事件:click、mousedown、mouseup、mousemove、mouseover、mouseout
    • 键盘事件:keydown、keyup、keypress
    • 表单事件:focus、blur、change、submit、reset
    • 自定义事件:自定义事件可以自定义一些事件，比如我们可以自定义一个叫做 myEvent 的事件，当某个元素触发了这个事件，我们就可以执行一些代码。

  4. 事件冒泡:当一个事件发生在元素上，它会先在这个元素上触发，然后逐级向上冒泡到父元素，直到遇到可以响应它的元素。

  5. 事件委托:当一个元素上有多个子元素，我们可以给父元素添加一个事件监听器，当子元素触发事件时，父元素可以监听到，然后执行相应的操作。

  6. 事件监听:addEventListener()可以为元素添加多个事件监听器，当事件发生时，会依次执行监听器中的代码。removeEventListener()可以移除事件监听器。
